# python-fundamentals
# 🚀 Python Fundamental Kursu: Öyrəndiklərim və Tapşırıqlar

Bu repozitoriyada Python proqramlaşdırma dilinə ilk addım atarkən öyrəndiyim əsas anlayışları, xətaları və yazdığım kodlarını toplayacam. Mənim üçün əsas olan  Python məntiqini oturtmaq baxımından bu tapşırıqlar əla bir təcrübə oldu.

---

### 1. Ədədlərlə işləmək və tiplərinin çevirilməsi
* `int` (tam ədəd) və `float` (kəsr ədəd) arasındakı fərqləri və `int()` və `float()` funksiyaları ilə dataları bir-birinə çevirilməsini anladım.

### 2. Mətnlər Üzərində işləmək 
* `.upper()`, `.lower()` və `.title()` metodları ilə mətnlərin görünüşünü dəyişdim. Mətn daxilindəki hər hansı bir sözü `.replace()` ilə başqa sözlə əvəzləməyi, `.find()` ilə sözün indeksini harada başladığını hesablamağı yerinə yetirdim. Cümləni sözlərə parçalamaq üçün `.split()`, parçalanmış sözləri aralarına simvol qoyub yenidən birləşdirmək üçün `.join()` metodlarının məntiqini başa düşdüm və yerinə yetirdim.


### 3. Data Strukturları və onların izahı

#### Lists `[]`
* `.append()` ilə sonuna element əlavə etməyi, `.pop()` ilə həm elementi silməyi, həm də silinən dəyəri ekranda göstərilməsini etdim. `.sort()` ( və `sorted()`  arasındakı o incə fərqin məntiqini möhkəmləndirdim. 

#### Dictionaries `{}`
* Key-Value (Açar-Dəyər) məntiqini və for key, value in dict.items():, .update()` vasitəsilə yeniləmə etməyi və lüğətdəki qiymətləri necə toplayacağımı qurdum.

#### Tuple `()` və Set `{}`
* **Tuple:** Elementlərinin silinə və ya dəyişdirilə bilməz (immutable) olduğunu, lazım gəldikdə isə onu `list()` edib hiylə işlədərək dəyişməyi etdim.
* **Set:** Bu strukturun sırasız olduğunu və ən əsası – daxilinə eyni elementdən ikinci dənəsini (dublikat) qəbul etmədiyini sınaqdan keçirdim.

###  4. Formatlama və Dövr Köməkçiləri
* `f-string` və `.format()` ilə dinamik cümlələr qurmağı vərdiş halına gətirdim.
